# TaskMaster Landing Page

This project is a responsive landing page for **TaskMaster**, a fictional productivity app, built using HTML and CSS. The goal is to showcase a clean, user-friendly design that highlights the app's features and encourages user engagement, while adhering to modern web development practices like semantic HTML and responsive design.

## Objective
The purpose of this project is to design and develop a responsive webpage for a fictional product or service using HTML and CSS. The page includes:
- A **Hero Section** to introduce the product with a subheadline and call-to-action (CTA) button.
- A **Features Section** with a three-column layout that adapts to different screen sizes.
- A **Footer Section** with contact details and copyright information.

## Features
1. **Hero Section**
   - Bold headline: "TaskMaster".
   - Subheadline: "Organize your life, boost your productivity, and achieve your goals."
   - CTA button: "Learn More" linking to the features section.

2. **Features Section**
   - Three-column layout with Flexbox, stacking into a single column on mobile.
   - Each column includes:
     - A placeholder icon (50x50px).
     - A feature title (e.g., "Task Management").
     - A short description of the feature.

3. **Footer Section**
   - Contact details: Email (`support@taskmaster.com`) and phone number (`+23480-5626-6468`).
   - Social media links (Twitter, Facebook, Instagram).
   - Copyright notice: "© 2023 TaskMaster. All rights reserved."

4. **Responsive Design**
   - Built with CSS Flexbox for a flexible layout.
   - Media queries ensure adaptability across devices (desktop, tablet, mobile).

## Technologies Used
- **HTML5**: For semantic structure and accessibility.
- **CSS3**: For styling and responsive design (Flexbox, media queries).
- No JavaScript is used, per project requirements.
